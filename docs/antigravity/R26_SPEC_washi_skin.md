# R26_SPEC_washi_skin (お正月UIスキン：和紙ミニマル 仕様書)

## 目的
Run1〜Run2.5の機能を維持したまま、UIを「和紙ミニマル」テーマへ刷新する。
生成り色の和紙背景、朱のアクセント、金の差し色、そして『謹賀新年』の透かし漢字を配し、お正月らしさと洗練された質感を両立させる。

## 配色設計 (CSS変数)
| 変数名 | 役割 | 想定値 (HEX) |
|---|---|---|
| `--washi-bg` | ベース背景 (生成り) | `#fcfaf2` (利休白茶に近い生成り) |
| `--washi-card` | カード背景 | `#ffffff` (純白すぎない白) |
| `--accent-red` | アクセント (朱) | `#b94047` (落ち着いた朱色) |
| `--accent-gold` | 差し色 (金) | `#d4af37` |
| `--ink` | 文字色 (墨) | `#333333` |
| `--muted` | 薄い墨/グレー | `#888888` |

## デザイン詳細

### 1. 背景 (和紙パターン)
- `styles.css` の `body` 背景を生成り色に。
- `assets/washi_pattern.svg` を作成し、ごく薄いノイズ/繊維感のあるテクスチャを表現。
- `background-repeat: repeat` で敷き詰める。

### 2. 透かし漢字『謹賀新年』
- `body::before` (右上) と `body::after` (左下) に配置。
- `content: "謹賀新年";`
- フォント：`serif` 系 (明朝体)
- `opacity: 0.07` 前後 (極薄)
- `position: fixed`, `pointer-events: none` で操作を妨げない。

### 3. カード (日付枠・おてつだい)
- `.day`, `.help-section`, `.sugoroku-section` 等の枠線を `1px solid var(--accent-red)` に。
- 背景は `var(--washi-card)`。
- box-shadow は控えめにし、フラットかつモダンな和風を目指す。

### 4. タイトルと金線
- `h1` の下に `2px solid var(--accent-gold)` の境界線を引く。
- タイトル表記：`🎍 だるまの にこにこカレンダー`

## 非破壊の約束
- `app.js` のロジック（状態管理、保存、リセット）は変更しない。
- 既存の「判子枠」スタンプ表示（CSS）と競合しないよう注意して上書き/調整する。
- 外部フォント（Google Fonts等）は使用せず、標準フォントスタックのみで構成。

## 検証計画
- UIの視認性（コントラスト）が保たれているか。
- 透かし文字がボタンなどの上にかぶってもクリックを邪魔しないか。
- P0ゲート（保存・リセット・保持）が正常か。
