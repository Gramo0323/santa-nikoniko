# A1: 症状・再現・期待値 (debug_helpPoints)

## 症状
- おてつだい（+1）ボタンを押してもポイント/進捗が増えない。
- ブラウザを更新（リロード）すると、おてつだいの進捗がリセットされる。

## 再現手順
1. アプリをブラウザで開く。
2. 「🧹 おてつだい」セクションの「+1」ボタンをクリックする。
3. ゲージや「スタンプ: 0こ」の表示に変化がないことを確認する。
4. 数回クリックした（つもりになった）後、ブラウザをリロードする。
5. 表示が初期状態（スタンプ: 0こ、あと5かい）に戻っていることを確認する。

## 期待される挙動
1. 「+1」ボタンをクリックすると：
   - ゲージ（○）が一つ変化する。
   - 5回クリックで「スタンプ」が+1され、ゲージがリセットされる。
   - localStorage に現在の進捗が保存される。
2. ブラウザをリロードすると：
   - localStorage から値が読み込まれ、リロード前の状態が復元される。
