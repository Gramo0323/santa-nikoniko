# R1_SPEC_daruma_ui (だるまUIパック仕様書)

## 目的
データ破壊を伴わずに、サンタUIからお正月用の「だるまUI」へ非破壊で移行する。

## 確定仕様
- **期間**: 2025/12/26 〜 2026/01/07 (13日間固定)
- **タイムゾーン**: Asia/Tokyo (JST) 固定
- **ヘッダー**:
  - タイトル: 🎍 だるまの にこにこカレンダー
  - サブコピー: 
    - 12/26〜12/31: 「もうすぐ おしょうがつ。きょうも にこにこ、ひとつずつ。」
    - 1/1: 「あけまして おめでとう。きょうも にこにこ、ひとつずつ。」
    - その他: 「きょうも にこにこ、ひとつずつ。」
- **通常スタンプ**: 円形の判子枠（朱肉色）。選択時に枠が太くなり、4px下へ沈む。
- **福バッジ**: JSTの「今日」のカードで、1かいめ・2かいめの両方が入力済みの場合のみ、右上に「福」バッジを表示。
- **導線A**: 12/26以降の初回起動時にのみ「おひっこし案内」を表示。テーマを自動的に daruma へ設定。

## 非破壊の範囲
- Supabaseの保存スキーマ、ログイン、おてつだい等のコアロジックは一切変更していない。
- 表示ロジックとCSSの微調整のみ。
