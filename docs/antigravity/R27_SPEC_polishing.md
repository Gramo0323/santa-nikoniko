# R27_SPEC_polishing (UI微調整 仕様書)

## 目的
Run 2.6の「和紙ミニマル」スキンを磨き上げ、より「お正月らしい上品さ」と「軽快な操作感」を両立させる。

## 調整内容

### 1. 和紙背景の質感アップ
- **ファイル**: `assets/washi_pattern.svg`
- **変更内容**: `feFuncA` の `slope` 値を `0.05` から **`0.07`** に増加。
- **狙い**: 和紙の繊維感を少し強め、生成り背景に深みを出す。

### 2. 操作フィードバック（押した感）の調整
- **ファイル**: `styles.css`
- **変更対象**: `.choice.selected`, `.choice:active`
- **変更内容**: `translateY(4px)` から **`translateY(2px)`** に減少。
- **狙い**: 「沈み込み」を控えめにすることで、現代的で軽快な操作感にする。

### 3. タイトル金線の箔押し感アップ
- **ファイル**: `styles.css`, `index.html`
- **変更内容**: 
    - `h1` の `border-bottom` を廃止。
    - `h1::after` 擬似要素で金線を実装。
    - 幅を `calc(100% - 40px)` 程度にし、左右に余白（マージン）を設ける。
- **狙い**: 線が端まで伸びないようにすることで、より「箔押し」に近い上品な印象を与える。

### 4. 🎍（タイトルマーク）に朱丸を追加
- **ファイル**: `index.html`, `styles.css`
- **変更内容**:
    - `index.html` の `🎍` を `<span class="title-mark">🎍</span>` で包む。
    - `.title-mark::after` で小さな朱丸（`var(--accent-red)`) を🎍の右上に配置。
- **狙い**: お正月らしいおめでたいアクセントを控えめに追加する。

## 透かし（謹賀新年）
- Run 2.6 のまま（変更なし）。

## 非破壊の約束
- `app.js` のロジックには一切触れない。
- `localStorage` のキー名・構造は変更しない。
- 既存の「判子枠」や「福バッジ」の視認性を損なわない。
- 外部フォント（Google Fonts等）は使用せず、標準フォントスタックを維持。
