# A1_SPEC_debug_calendarFrame.md

## 症状
- ログインに成功した後、カレンダーの「日付」「1かいめ/2かいめ」の枠（#days 要素の中身）が全く表示されない。
- スタンプやおてつだい表示（上のセクション）は残っている。
- リロードしてもログイン状態が維持されている限り、枠は表示されない。

## 期待値
- ログインの有無に関わらず、カレンダー枠（日付とスタンプボタン）が常に表示されること。
- データロード完了後、保存されているスタンプの状態が反映されること。

## 再現条件
- **ログイン状態**: ログインに成功し、Supabaseからデータが取得された際に発生する。
- **未ログイン状態**: 発生しない（LocalStorageからのロードでは枠が表示される）。

## 調査対象環境
- ローカル開発環境 (http://localhost:8000)
- ブラウザ: Chrome / Firefox 等 (JavaScriptが動作する環境)
