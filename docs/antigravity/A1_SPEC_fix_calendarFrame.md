# A1_SPEC_fix_calendarFrame.md

## 概要
ログイン成功時にカレンダー枠（#days）が表示されない不具合を修正し、あらゆるロード条件下でカレンダー枠が確実に表示されるようにします。

## 完了条件 (DoD)
1.  **未ログイン時の表示保証**: 起動直後、データの同期を待たずにカレンダー枠（日付/1かいめ/2かいめ）が表示されること。
2.  **ログイン成功時の表示保証**: Supabaseからのデータロード成功後も、カレンダー枠が表示され続ける（または再描画される）こと。
3.  **ログイン失敗時の表示保証**: Supabaseロード失敗時でも、LocalStorageのデータを用いて枠が表示されること。
4.  **機能維持**: 既存のスタンプ操作、おてつだいカウント、およびそれらの保存機能が正常に動作すること。

## 非ゴール
- カレンダー枠の配置変更（移動）
- CSSによるデザイン変更
- Supabaseのスキーマ変更
- 大規模なリファクタリング
